주요 git 명령어   
- add : 파일을 추적 대상으로 포함시킬 때, 또는 커밋 대상으로 포함시킬 때 사용
  - 예) git add   
- commit   
- git reset HEAD : stage된 파일을 unstaged로 변경   
- git checkout -- : stage되어 있는 파일을 수정한 후 수정 전으로 되돌림
- branch
- merge
- status
- log
  - 예) git log --oneline --decorate --graph --all
